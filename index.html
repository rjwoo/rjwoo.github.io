Workout Web Application
Spotter
View Source Code - Visit Spotter

Spotter is a web application that allows user to track their progress in the gym, by rendering weight and volume progress. The project's back-end uses Ruby on Rails and a PostgreSQL database, with GitHub for version control. The front-end is a combination of HTML, CS, Bootstrap and jQuery, along with two libraries: Unicorn UI and Chart.js.

Personal Management Tool
Manage Me
View Source Code - Visit Manage Me

Manage Me is a personal management tool that allows users to manage their projects. The app's front-end is comprised of HTML, CSS, Bootstrap, and jQuery. On the back-end, I used Ruby on Rails with a PostgreSQL database. Some of the gems included were Simple Form, Cancancan, and Faker.

72-Hour Hackathon Group Project
CodeCore For Hire
View Source Code - Visit CodeCore For Hire

CodeCore For Hire allows employers to browse a functional database filtering CodeCore graduates seeking work opportunities. The project was created over the span of 72-hours with 6 other colleagues using Github for version control. The app uses a combination of CSS, JavaScript, jQuery and AJAX for the front-end including the button features and the student profile pages. We used Ruby on Rails for the back-end incorporating a PostgreSQL database.
